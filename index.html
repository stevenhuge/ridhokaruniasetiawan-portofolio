Selama tiga tahun, saya mengikuti berbagai pelatihan dari tingkat pemula hingga mahir. Selain itu, saya berhasil memperoleh beberapa sertifikasi, termasuk sertifikasi BNSP sebagai Junior Web Developer dan sertifikasi dari perusahaan IT di Surakarta.

Pengalaman saya juga mencakup magang di Universitas Gadjah Mada, tepatnya di Sekolah Vokasi, di mana saya bekerja sebagai Android Developer dan Frontend Web Developer selama enam bulan. Saat ini, saya sedang menempuh pendidikan S1 Sistem Informasi di Universitas Amikom Yogyakarta. Tujuan saya adalah untuk meningkatkan dan memperdalam pengetahuan saya di bidang IT, serta mengeksplorasi minat saya dalam bisnis IT.
